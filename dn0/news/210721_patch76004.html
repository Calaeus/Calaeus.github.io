News: MXOEMU Patch 7.6004

Items on this page are part of patch 7.6004.

Client Changes

  • Music no longer reduces its volume on resume after an Alt-Tab
  • All available in-game organizations can now be used if needed
  • Radio favorites now work properly, add them with /radio addfave name|URL
  • Fix navmesh breaks in Mara NW, Rogers Way, large01 construct (includes moving the restaurant down)
  • Fix some instances of floating trash, props, and billboards being inset into their buildings
  • The sentinel newspaper vendors, help, and cinematics now work properly from the in-game browser
  • Various other small bug fixes

Launcher Changes

  • Placing portable.txt inside MxO's install dir will make it store all settings and user data within that folder (portable install)
  • UAC Administrator elevation will automatically be requested if it is needed, there is no need to do it manually anymore
  • Improve Linux compatibility under Wine
